# Plant Dojo Website Starter

A clean beginner-friendly website starter you can upload to GitHub and publish with GitHub Pages.

## Files

- `index.html` — main website content
- `styles.css` — design, colors, layout, and mobile responsive styling
- `script.js` — mobile menu and automatic footer year

## How to upload to GitHub

1. Go to GitHub and create a new repository.
2. Name it something like `plant-dojo-website`.
3. Upload these files into the repository.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
6. Save.
7. GitHub will give you a live website link.

## How to edit

Open `index.html` and change the text inside each section.

Open `styles.css` to change colors, spacing, fonts, and layout.
